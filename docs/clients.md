## Connection Settings

Use these settings in your client:

| Setting  | Value                                |
| -------- | ------------------------------------ |
| Server   | Your EDGE server IP                  |
| Port     | 443 (SS) / 80 (HTTP) / 1080 (SOCKS5) |
| Method   | `aes-256-gcm`                        |
| Password | From `./xcp.sh user ls`              |
