# Xray Chain Proxy

> **نسخه 2.0.0** | [تغییرات](changelog.md)

پروکسی زنجیره‌ای برای دور زدن سانسور اینترنت، بر پایه [Xray-core](https://github.com/XTLS/Xray-core).

## معماری

```
Client --> EDGE --> GATEWAY --> Internet
```

| سرور      | نقش                                          |
| --------- | -------------------------------------------- |
| `GATEWAY` | نود خروجی با اینترنت آزاد                    |
| `EDGE`    | نود ورودی، ترافیک را به GATEWAY ارسال می‌کند |

## امکانات

- چند پروتکلی: Shadowsocks، HTTP، SOCKS5
- معماری زنجیره‌ای با چند هاپ
- مدیریت کاربران با QR کد
- آمار ترافیک هر کاربر
- بروزرسانی خودکار

## پروتکل‌ها و پورت‌ها

| پروتکل      | پورت | توضیحات                     |
| ----------- | ---- | --------------------------- |
| Shadowsocks | 443  | رمزنگاری شده، شبیه HTTPS    |
| HTTP        | 80   | پروکسی HTTP استاندارد       |
| SOCKS5      | 1080 | همه‌منظوره، پشتیبانی از UDP |

همه پروتکل‌ها از یک نام کاربری/رمز عبور استفاده می‌کنند.

## پیش‌نیازها

- سرور Debian/Ubuntu با دسترسی root
- سرور GATEWAY با اینترنت آزاد
- سرور EDGE در دسترس کاربران

## فایل‌ها

| فایل    | مسیر                          |
| ------- | ----------------------------- |
| باینری  | `/usr/local/xray/xray`        |
| تنظیمات | `/usr/local/xray/config.json` |
| لاگ‌ها  | `/var/log/xray/`              |
| کش      | `/tmp/xray-cache/`            |

## وابستگی‌ها

**الزامی:** `curl` `unzip` `jq`

**اختیاری:** `qrencode` (QR کد)، `speedtest-cli` (تست سرعت)
